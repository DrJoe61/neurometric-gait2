# neurometric-gait2 — clean package

This folder is a ready-to-upload version of your NeuroGait project for
https://github.com/DrJoe61/neurometric-gait2

## What's inside
- Full Android app source (app/, gradle/, build files) — your real, buildable code
- docs/ — your 4 project documents (guide, sensor-fusion notes, summary, .docx)
- libs/ — the Movesense mdslib .aar dependency
- .gitignore — keeps build artifacts OUT of GitHub (fixes the previous mess)

## What was removed vs. the messy upload
Build/IDE junk that never belongs in GitHub: workspace.xml, local.properties,
last-build.bin, outputFiles.bin, *.apk, .gradle/ and build/ caches, gc.properties,
buildOutputCleanup.lock, cache.properties, file-system.probe, problems-report.html,
manifest-merger-debug-report.txt, and the .idea/ IDE folder.
